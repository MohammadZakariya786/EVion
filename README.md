## âš¡ Getting Started

### Prerequisites

* Node.js (v14 or higher)
* npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/MohammadZakariya786/EVion.git
cd evion
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

### Running the App

```bash

```
